# Credit Card Fraud Detection System 🚀

This is a Machine Learning web app built using Streamlit that predicts fraudulent credit card transactions.

## Features
- Predicts fraudulent transactions
- Interactive UI for input data
- Real-time prediction using trained ML model
- Deployed using Streamlit Cloud

## Tech Stack
Python, Streamlit, Pandas, NumPy, Scikit-learn, Plotly, SHAP

## Setup
This project requires **Python 3.11** (scikit-learn 1.3.2 does not have prebuilt installers for newer Python versions like 3.12+).

```bash
git clone https://github.com/JayaPrathiba/Credit-Card-Transaction-Fraud-Detection-System.git
cd Credit-Card-Transaction-Fraud-Detection-System
python -m venv venv
venv\Scripts\activate    # On Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
